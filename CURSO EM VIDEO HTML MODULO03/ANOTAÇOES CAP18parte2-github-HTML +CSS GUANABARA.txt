Hospedando no github
Com GitHub Pages.
Clique em settings no repositorio.
procure por git hub pages.
primeiro clique em choose a theme.
e em seguide clique em commit.
assim voce vai ter seu endereço
exemplo:
https://sandro2795.github.io/Curso-html5-css3/

criando um repositorio para abrir diretamente seu site:
copie e cole os arquivo e o projeto e o css. 
em um diretorio criado no github desktop.
em seguida mude o nome do projeto para index
E em seguida vá no repositorio que ficou no site do

selecione  NONE e clique em main branch
ou a branch criado para o repositorio.
assim será possivel acessar seu projeto por um endereço
